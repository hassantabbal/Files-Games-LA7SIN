How to install PKG files on PS3 (safe method)

  1. Ensure your PS3 is compatible and has enough free space.
  2. Download all PKG parts + LICENSE/patch files.
  3. Copy files via FAT32 USB or FTP.
  4. Install via package installer on PS3.
  5. Reboot console after applying patches.
